## 🧭 How to embed the AI ChatBot on any website (summary)
1. The ChatBot widget is a small JS snippet + backend endpoint (OpenAI / Cloud AI).  
2. Offer theme options (CSS variables) and voice (Web Speech API) support.  
3. Response delay settings: call the bot with simulated delay or queue the reply (10s / 30s / 60s).  
4. Provide client-side snippet:
```html
<!-- example embed -->
<div id="tdm-chat-widget"></div>
<script src="https://cdn.example.com/tdm-chat-widget.js"></script>
<script>
  TDMChat.init({
    target: '#tdm-chat-widget',
    theme: 'dark',
    voice: true,
    responseDelay: 10, // seconds: 10 | 30 | 60
    apiEndpoint: 'https://api.yoursite.com/chat'
  });
</script>
